# The Sola Story - React + Vite

## Run

1. `npm install`
2. `npm run dev`

## Assets

For Vite, place your images in `./public/assets` using these names:

- `product-bottle.jpg` (hero + product section)
- `fatema-portrait.jpg` (story section)
- `product-label.jpg` (label section)

Then restart dev server (`npm run dev`) if it was already running.
